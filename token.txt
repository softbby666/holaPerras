Create Personal Access Token on GitHub
From your GitHub account, go to Settings => 
                      Developer Settings => 
                      Personal Access Token => 
                      Generate New Token (Give your password) => 
                      Fillup the form => 
                      click Generate token => 
                      Copy the generated Token.
